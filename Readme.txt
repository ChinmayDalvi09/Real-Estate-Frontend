# EstateAgency00 🏡

A full-featured real estate web application built with Django. This project supports:

- 🧑‍💼 Role-based user authentication (Admin, Buyer, Seller)
- 🏘️ Property listings with image uploads
- 🔍 Search and filter by price, location, and type
- 📩 Inquiry and messaging system
- 📅 Booking and appointment management

## Tech Stack

- **Backend**: Django, Python
- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **Database**: MySQL

## Setup

1. Clone the repo
2. Set up a virtual environment
3. Install dependencies
4. Configure MySQL database
5. Run migrations and start the server
